

# Search an Element in a Linked List


### Problem Statement

Given the head of a linked list and an integer value `val`, determine whether `val` exists in the linked list.
Return `true` if found, otherwise return `false`.

### Examples

Example 1:
Input:

```
0 -> 1 -> 2
val = 2
```

Output:

```
true
```

Example 2:
Input:

```
12 -> 5 -> 8 -> 7
val = 6
```

Output:

```
false
```

### Approach

To search for a value in the linked list, traverse from the head to the end and compare each node’s data with the target value.

### Algorithm

Step-by-step process:

1. Create a pointer `current` pointing to `head`.
2. While `current` is not NULL:

   * If `current->data == val`, return `true`.
   * Otherwise move to the next node: `current = current->next`.
3. After full traversal, return `false`.

### Dry Run Example

Linked List:

```
10 → 20 → 30 → NULL
val = 20
```

| current node value | comparison | result      |
| ------------------ | ---------- | ----------- |
| 10                 | 10 != 20   | continue    |
| 20                 | 20 == 20   | return true |

Answer: `true`

### Fully Commented C++ Code

```cpp
#include <bits/stdc++.h>
using namespace std;

// Node structure for singly linked list
struct Node {
    int data;
    Node* next;

    // Constructor initializes node with value and next = NULL
    Node(int val) : data(val), next(NULL) {}
};

class Solution {
public:
    // Function to search for a value in the linked list
    bool searchValue(Node* head, int key) {
        Node* current = head;  // start traversal from head

        // Traverse until end of the list
        while (current != NULL) {
            // Check if current node contains the key
            if (current->data == key) {
                return true;  // value found
            }
            current = current->next;  // move to next node
        }

        // After complete traversal, value not found
        return false;
    }
};

// Driver code
int main() {
    // Creating linked list: 10 -> 20 -> 30
    Node* head = new Node(10);
    head->next = new Node(20);
    head->next->next = new Node(30);

    Solution obj;

    // Search for a value
    int val = 20;
    bool found = obj.searchValue(head, val);

    if (found) cout << "Found\n";
    else cout << "Not Found\n";

    return 0;
}
```

### Output

```
Found
```
